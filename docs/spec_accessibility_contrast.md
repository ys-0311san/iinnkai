# 仕様書：視認性（コントラスト）ベストプラクティス対応

## 概要

「薄い背景色の上に薄い文字色」という問題が複数箇所で確認されている。
WCAG AA 準拠（通常テキスト 4.5:1 以上）を目標に、テキスト系 `rgba` の透明度を引き上げる。

**対象外（変更しない）:** ボーダー、ボックスシャドウ、セパレーター線など、テキスト以外の装飾要素の `rgba`

---

## 修正ルール（全体方針）

| 用途 | 変更前 alpha の目安 | 変更後 alpha |
|---|---|---|
| 補助テキスト（hint / help-text 等） | 0.4〜0.5 | **0.68** |
| ナビ・サイドバー 非アクティブアイテム | 0.65〜0.75 | **0.82** |
| 一般テキスト・ラベル | 0.7 | **0.87** |
| 茶系アクセント（Q./A. マーカー等） | rgba(139,111,71,0.7〜0.8) | **var(--cedar)** （透明度なし） |
